A web-based reading intervention app inspired by Savvas I-LIT, targeting grades 4-8 struggling readers.

## Core Experiences

### Interactive Reader
Adaptive nonfiction passages with slide-based navigation, embedded comprehension checkpoints (highlight, drag-and-drop), audio support, and summary writing with feedback.

### Digital Library
Browsable collection of leveled fiction and nonfiction texts with a book-style reader, annotations, text-to-speech, and progress tracking.

## Tech Stack
- Next.js 15 (App Router, TypeScript)
- Tailwind CSS
- Framer Motion
- localStorage for student progress
- Web Speech API for TTS
- Vercel deployment

## Status
Research & design phase. See `/docs/` for full UI documentation and implementation plan.
